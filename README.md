# 🍦 DeliveryDash

A 2D top-down delivery game built in **Unity**, where you drive an ice-cream van around a neighborhood, pick up packages, and deliver them straight to the doorstep!

This project was built while following a **Udemy course** as a learning exercise in core Unity 2D game development concepts.

---

## 📖 Game Description

You control a delivery van that drives through a cozy, top-down village map. The objective is simple:

1. **Pick up** packages scattered around the map.
2. **Navigate** through streets and obstacles.
3. **Deliver** the packages to the correct doorstep / floor marker.

Along the way, you'll encounter obstacles, collectible boost orbs, and other interactive elements that add a bit of challenge and fun to the delivery run.

## 📚 What I Learned

This was a learning-focused project, and through building it I practiced:

- **Transforms & Rotation** — Moving and rotating the van smoothly based on player input.
- **Basic Input System** — Reading player input using Unity's modern Input System.
- **Prefabs** — Creating reusable objects (packages, obstacles, boost orbs) for efficient level design.
- **Basic C# Scripting** — Writing gameplay logic, movement scripts, and collision handling.
- **`Time.deltaTime`** — Making movement and animations frame-rate independent.

---

## 🚀 How to Run

1. Clone or download this repository.
2. Open the project folder in **Unity Hub** (recommended version: Unity 6 / 6000.2.x or later).
3. Open the `SampleScene` from the Project window.
4. Press the **Play** button to start the game.
